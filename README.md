# SAPZ Project Interactive Map

Live interactive map displaying 821 land parcels for the Special Agro-Industrial Processing Zones (SAPZ) Climate-Smart Agriculture project in Kano State, Nigeria.

## Features

- **821 mapped land parcels** across 10 communities
- **354.41 hectares** of agricultural land
- Interactive parcel information with detailed beneficiary data
- Multiple map layers and visualization options
- Fully responsive and mobile-friendly
