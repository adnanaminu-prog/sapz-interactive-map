About This Map

This interactive map displays the land parcels of 821 smallholder farmers enrolled in the Special Agro-Industrial Processing Zones (SAPZ) Program. Click on any parcel to view detailed information about the beneficiary, including demographics, crop types, land tenure, and access to services. Use the layer control to filter by gender, community, or view all parcels.
